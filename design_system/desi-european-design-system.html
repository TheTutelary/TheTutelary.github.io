Design System Reference

A visual bridge between Indian warmth and Scandinavian Lagom.

Design Principles

Lagom First

"Just enough." We strip away the noise to let the storytelling breathe. Minimalism with purpose.

Radical Warmth

Clean lines don't have to be cold. We use Cardamom Cream and Saffron to keep the soul alive.

Editorial Flow

The website should feel like a premium travel magazine. Typography is our primary interface.

The Lagom Guidelines

DO
  • Use Cardamom Cream for large background areas.
  • Keep line height generous (1.75) for long stories.
  • Use Saffron Sunrise strictly for interactive elements.
DON'T
  • Use pure black (#000) for body text. Use Deep Earth.
  • Add complex gradients or blurs. Keep it grounded.
  • Forget the "Editorial Duo" pairing in components.
