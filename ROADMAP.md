# Desi European — Strategic Roadmap 🚀

This roadmap outlines the long-term vision for transforming the **Desi European** Travelogue into a world-class, design-led editorial platform.

## 🎨 1. Universal Design System (Top Priority)
Before scaling content further, we will unify the visual language across all 15+ journeys.

- [ ] **Semantic Token Migration**: Move from ad-hoc hex codes to a centralized token system in `assets/css/variables.css`.
- [ ] **Typography Scale**: Define a standard hierarchy for all headings (H1–H4), pull-quotes, and captions.
- [ ] **Component Library**: Create reusable HTML/CSS blocks for "Quick Facts" (weather, duration, cost), interactive galleries, and map pins.
- [ ] **Dark Mode Support**: Implement a subtle dark mode for night-time reading of long-form stories.

## 📄 2. Page-by-Page Improvement Audit
We will perform a deep dive into every core page to refine the UI/UX and narrative depth.

### 🏠 Homepage (`index.html`)
- [ ] **Interactive Progress Map**: Add a visual tracker of the 12 countries.
- [ ] **Trip Categories**: Filter by "Foodie Guides," "Slow Travel," or "Family Adventures."
- [ ] **Hero Splendor**: Optimize the hero image for faster loading and implement a subtle fade-in animation.

### 📖 About Us (`about.html`)
- [ ] **Journey Timeline**: Replace static text with an interactive "India-to-Sweden" timeline.
- [ ] **Our Philosophy**: A dedicated section on "The Art of Slow Travel" and "Lagom" living.

### ✍️ Blog Post Architecture (`template.html`)
- [ ] **Reading Progress Indicator**: Add a progress bar for long-form content.
- [ ] **Image Lightbox**: Implement pure CSS or low-weight JS lightbox for high-res galleries.
- [ ] **Deep-Linking**: Ensure each heading has an anchor for easy sharing of specific story segments.

## 🛠 3. Technical Scale & Performance
- [ ] **Jekyll Migration**: Automate the grid generation and use YAML for metadata.
- [ ] **Image Pipeline**: Automate WebP conversion for all assets.
- [ ] **SEO Pass**: Finalize descriptions and semantic headers for all 15 pages to improve search visibility.
